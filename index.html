SQL Roadmap for Data Analysts
Exactly kitna SQL aana chahiye — phase by phase
PHASE 1 Foundation — Must Know 2–3 weeks
SELECT, WHERE, ORDER BY Filtering with AND/OR/IN/BETWEEN GROUP BY + HAVING COUNT, SUM, AVG, MIN, MAX DISTINCT NULL handling (IS NULL, COALESCE) LIKE / wildcards LIMIT / TOP
PHASE 2 Intermediate — Core Analyst Skills 3–4 weeks
JOINS — INNER, LEFT, RIGHT, FULL Subqueries (nested SELECT) CTEs (WITH clause) CASE WHEN String functions (TRIM, UPPER, CONCAT) Date functions (DATEDIFF, DATEPART) UNION / UNION ALL
PHASE 3 Advanced — Interview Level 3–5 weeks
Window Functions — ROW_NUMBER, RANK DENSE_RANK, NTILE LAG / LEAD PARTITION BY Running totals / Moving averages Self Joins Performance basics (INDEX idea)
PHASE 4 Real World Practice Ongoing
LeetCode SQL 50 StrataScratch DataLemur Mode Analytics tutorials Build your own mini-projects Mock interviews
💡 Pro Tip: Data Analyst interviews mein Window Functions aur CTEs sabse zyada pooche jaate hain. Agar tum RANK(), DENSE_RANK(), LAG/LEAD confidently likh sako — tum 80% candidates se aage ho. Pehle logic samjho, syntax baad mein aata hai.

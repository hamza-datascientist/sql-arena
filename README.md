# 🎯 SQL Arena — Data Analyst Practice Platform

An AI-powered SQL practice platform built for aspiring Data Analysts. Practice real interview questions, get instant AI feedback on your queries, and track your learning progress.

![HTML](https://img.shields.io/badge/Built%20With-HTML%2FJS-ff6b35?style=for-the-badge)

---

## ✨ Features

- **📍 Roadmap** — Phase-wise SQL learning path for Data Analysts with timelines
- **⚡ Practice** — 6 real interview-style SQL questions (Easy → Hard) with AI assessment
- **🎯 Interview Prep** — 12+ common interview Q&As with SQL code examples
- **📊 Skill Assessment** — Describe your experience, get personalized AI feedback and next steps

```bash
# Option 2: VS Code Live Server
Right-click → Open with Live Server
```

---

## 🌐 Deploy on GitHub Pages

1. Create a new GitHub repository (e.g., `sql-arena`)
2. Upload `sql-practice.html` and `README.md`
3. Go to **Settings → Pages**
4. Set Branch to `main` → Click **Save**
5. Your site will be live at `https://your-username.github.io/sql-arena`
